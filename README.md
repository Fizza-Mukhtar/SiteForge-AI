# SiteForge AI

AI-powered full-stack website builder that generates professional websites from natural language prompts.

## Features
- AI-generated website layouts and content
- Modern SaaS UI with dark mode
- Secure server-side AI processing
- Responsive design

## Tech Stack
- React
- TypeScript
- Next.js (App Router)
- Tailwind CSS
- Google Gemini

## Setup
```bash
npm install
npm run dev
